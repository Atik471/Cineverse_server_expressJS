# Movie Streaming API (Backend)

Live Site URL: [Cineverse server](https://cineverse-9wce3xir4-atikur-rahmans-projects-e1054064.vercel.app/)

## Features:
- **Create Movie Entries**: Add new movie data with details such as title, genre, poster URL, release year, and rating.
- **Fetch Movies**: Retrieve movies based on filters like genre, year, and rating.
- **Update Movie Information**: Edit movie details like summary, duration, and rating.
- **Delete Movies**: Remove movies from the database.
- **User Authentication**: Implement secure user authentication using JWT (JSON Web Tokens).

## Technologies Used:
- Node.js
- Express
- MongoDB (or any database you're using)
